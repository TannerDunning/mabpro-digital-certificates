=== MABPRO-DIGITAL-CERTIFICATES ===

This plugin creates a custom certificate feature on a WordPress website that allows instructors to send personalized digital certificates to their students' emails. Instructors can purchase a set number of certificates through a WooCommerce product and then use a form to enter the students' information and send the certificates. The plugin includes a form builder, PDF generation, and email sending functionality.

== Description ==
The purpose of this plugin is to provide a custom certificate feature to your WordPress website that enables your instructors to send personalized digital certificates to their students' email addresses. It provides a form builder that lets your instructors to input students' details and then send the certificates. The plugin includes PDF Generation and Email Sending functionality.

== Installation ==

Upload the mabpro-digital-certificates folder to the /wp-content/plugins/ directory
Activate the plugin through the 'Plugins' menu in WordPress
== Frequently Asked Questions ==
Q: How is the plugin used?
A: The plugin is used by Instructors to send personalized digital certificates by creating a form with student data.

Q: How can I add the certificate form to my website?
A: To add the form, you need to insert the shortcode [certificate_form] in a page of your preference.

Q: How does the certificate look like?
A: The certificate has a built-in design template, though, you can customize to your taste using the form builder.
